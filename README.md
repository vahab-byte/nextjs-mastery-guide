# Next.js Mastery Guide

## Project Info

This is a comprehensive guide and platform for learning Next.js and React.

## Getting Started

1.  **Install dependencies**:
    ```sh
    npm install
    ```

2.  **Start the development server**:
    ```sh
    npm run dev
    ```

## Technologies

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS
-   Supabase
